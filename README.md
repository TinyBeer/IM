# ChatRoom

海量用户聊天系统  -----Go语言网络编程的实践

项目目的：实现基于Golang开发多人在线聊天平台

相关技术：Go socket Redis Vue ase



项目包括web客户端   服务端两部分

目前正在进行服务端开发。。。

目前功能
终端
1.用户登录 用户注册
2.用户上线 下线通知 在线用户列表
3.消息群发  点对点聊天等功能
web
1.用户登录
2.群聊收发
